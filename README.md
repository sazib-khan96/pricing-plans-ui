# Pricing Toggle Component

A responsive pricing component built with HTML, CSS, and JavaScript. It includes monthly and yearly pricing plans along with a dark and light mode toggle.

## Features

* Responsive design for desktop and mobile devices
* Monthly and yearly pricing switch
* Dark and light mode toggle
* Clean and modern UI
* Built with vanilla HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
